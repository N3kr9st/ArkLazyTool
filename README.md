# ArkLazyTool

**ArkLazyTool** ist ein praktisches Tool für den ambitionierten Züchter der schnell das max Level seiner zukünftigen Babys wissen möchte.

## Voraussetzungen

- **Windows-Betriebssystem**
- .NET Framework 4.8 (oder neuer)
